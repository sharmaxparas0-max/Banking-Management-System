# Banking Management System

A Java-based Banking Management System developed using JDBC and MySQL. The application enables users to perform core banking operations through a console-based interface.

## Features

* Create bank accounts
* User authentication
* Deposit money
* Withdraw money
* Check account balance
* Fund transfers
* Transaction management
* Database integration using JDBC and MySQL

## Technologies Used

* Java
* JDBC
* MySQL
* Object-Oriented Programming (OOP)

## Project Structure

* BankingApp.java – Main application entry point
* User.java – User management
* Accounts.java – Account operations
* AccountManager.java – Banking services and transactions

## Learning Outcomes

* JDBC database connectivity
* SQL query execution
* Java OOP concepts
* Exception handling
* Database-driven application development
